**Key Technical Decisions:**
1. **Tailwind v3.4.17 over v4** - v4 just released (January 2025) with breaking changes, v3 is stable and widely-used
2. **Vite + Vanilla TypeScript over React** - Simpler for type calculator logic, matches mockup patterns
3. **Responsive web layout** - Desktop top navigation + mobile bottom navigation (NOT mobile mockup in center of screen)
4. **Progressive Web App approach** - Cross-platform without app store approval

**Errors Encountered and Resolved:**
1. **Tailwind v4 PostCSS Error** - Initially installed v4.1.17, which uses separate `@tailwindcss/postcss` plugin. Researched best practices, downgraded to v3.4.17 for stability.
2. **NPM Permission Error** - Used `rm -rf node_modules package-lock.json` to resolve WSL2 file permission issue.
